# AI Course Finder 🚀

Turn YouTube noise into structured AI-powered learning paths.

AI Course Finder is a modern web application that helps users discover the best courses, tutorials, and problem-solving videos using **Gemini AI** and the **YouTube API**. Instead of endlessly scrolling through YouTube, users can describe what they want to learn or the issue they are facing, and the AI intelligently recommends the most relevant content.

---

## ✨ Features

### 🔍 Smart Course Discovery

Search any skill or technology and receive:

* Curated YouTube courses
* Structured learning paths
* Beginner to advanced recommendations
* Recent and high-quality content

### 🧠 AI-Powered Problem Solving

Describe your issue in plain English and the AI will:

* Analyze the problem
* Understand the learning intent
* Fetch the most relevant tutorials and solutions from YouTube

### 📚 Structured Learning Experience

* Personalized recommendations
* Smart search filtering
* Playlist-focused learning
* Reduced tutorial overload

### ⚡ Modern UI/UX

* Fully responsive design
* Dark futuristic interface
* Smooth and clean user experience
* Built with Tailwind CSS

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/MdFoysalHossain/AiCourseFinder.git
```

---

## 2. Navigate to the Project Folder

```bash
cd ai-course-finder
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Create Environment Variables

Create a `.env` file in the root directory:

```env
VITE_GEMINI_API_KEY=your_gemini_api_key
VITE_YOUTUBE_API_KEY=your_youtube_api_key
```

---

## 5. Start Development Server

```bash
npm run dev
```

---

# 💡 How It Works

## Skill Learning Mode

Users enter a skill they want to learn.

Example:

```txt
Learn MERN Stack Development
```

The AI:

1. Understands the learning goal
2. Searches YouTube API
3. Filters quality courses
4. Returns the best structured recommendations

---

## Problem Solving Mode

Users describe their issue naturally.

Example:

```txt
My React useEffect keeps causing infinite re-renders
```

The AI:

1. Analyzes the problem
2. Extracts technical context
3. Finds relevant solution videos
4. Suggests targeted tutorials
